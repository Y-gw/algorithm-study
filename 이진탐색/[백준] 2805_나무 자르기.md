### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    start 길이 1, end 길이 max(height)로 놓고 이분탐색을 반복.
    cnt 변수를 통해 필요한 나무의 길이에 알맞게 절단 높이를 조정.

### 코드의 시간복잡도와 그 이유:
    n개의 값 입력 -> O(n)
    이분 탐색 -> O(log(n))
    높이 비교 -> O(n)
    O(n+log(n))의 시간복잡도를 가짐.


### 코드:
```python
n, m = map(int, input().split())
height = list(map(int, input().split()))

start = 1
end = max(height)

while start <= end:
    mid = (start + end) // 2
    cnt = 0
    
    for i in height:
        if i > mid:
            cnt += i - mid

    if cnt < m:
        end = mid - 1
    else:
        start = mid + 1

print(end)
```
