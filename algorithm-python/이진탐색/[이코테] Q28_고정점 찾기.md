### 문제풀이 핵심아이디어 또는 새롭게 알게된 내용: 
    고정점 : 값과 인덱스가 동일한 원소.

### 코드의 시간복잡도와 그 이유:
    n개의 수열 입력 -> O(n)
    이분 탐색 -> O(log(n))
    O(n+log(n))의 시간복잡도를 가짐.


### 코드:
```python
n = int(input())
seq = list(map(int, input().split()))

start = 0
end = len(seq)
result = -1

while start <= end:
    mid = (start + end) // 2

    if seq[mid] == mid:
        result = mid
        break

    elif seq[mid] > mid:
        end = mid - 1

    else:
        start = mid + 1
    
print(result)
```
